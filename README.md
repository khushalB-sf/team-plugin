## Quick Reference

```bash
# Open the plugin browser
/plugin

# Discover available plugins
/plugin discover

# Install a plugin
/plugin install commit-commands@claude-plugins-official

# List installed plugins
/plugin list

# Remove a plugin
/plugin remove commit-commands@claude-plugins-official
```
